Html Semantics Tags

Form

A semantic tag to take multiple forms of input from user.

The form element can contain one or more of the following form elements

There are multiple kinds of element to take input from user.

  • input
  • textarea
  • button

Table

The table tag is used to define a table.

The table tag contains other tags which is used to define structure of a table.

HTML tables are used to manage the layout of the page e.g. header section, navigation bar, body content, footer section etc.

It is recommended to use div tag over table to manage the layout of the page .

The important tags of table are-

  • tr
  • th
  • td

Article

The content of article tags are self-contained and idependent.

An article should have its own meaning and be easily differentiated from the rest of the web page content.

The place where article element can be used are-

  • user comment
  • blogs articles
  • newspaper content

Section

Section tags is used to define the section of documents.

It is used when we need two headers or footers .

It is udes to distribute the content.

The main advantage of the section tag is, it is a semantic element, which describes its meaning to both browser and developer.

Aside

The aside tag is used to describe the main object of the web page in a shorter way like a highlighter.

It basically identifies the content that is related to the primary content of the web page but does not constitute the main intent of the primary page.

The aside tag contains mainly-

  • author information
  • links
  • related content

Time

The time tag is mainly used to display the human readable date or time.

The main benefit of time tag is that they can offer to add birthday reminders or scheduled events.

The time element isn’t particularly useful until paired with a datetime attribute.

One a datetime attribute has been associated with the time element, the browser knows a lot more about the time described in the time element and can use that information when interacting with other applications — such as when importing information from an HTML email or webpage into a calendar application.

Forming a new habit

Dr. Maltz found that it would take a patient about 21 days to get adjusted to the new situation.

This all forced dr. maltz to think about his own adjustment period to new behaviour and noticed that he was also taking nearly 21 days to form a new habit.

Many people tried and chose habits like drinking a bottle of water with lunch or running for 15 minutes before dinner.

At the end of 12 weeks ,the researchers analysed the data to determine how long it took each person to go from starting a new behaviour to automatically doing it.

First, make a decision. Decide clearly that you are going to begin acting in a specific way 100% of the time, whenever that behavior is required.

For example, if you decide to arise early and exercise each morning, set your clock for a specific time, and when the alarm goes off, immediately get up, put on your exercise clothes and begin your exercise session.

Second, never allow an exception to your new habit pattern during the formative stages. Don’t make excuses or rationalizations. Don’t let yourself off the hook.

If you resolve to get up at 6:00 AM each morning, discipline yourself to get up at 6:00 AM, every single morning until this becomes automatic.

Third, tell others that you are going to begin practicing a particular behavior.

It is amazing how much more disciplined and determined you will become when you know that others are watching you to see if you have the willpower to follow through on your resolution.

Fourth, visualize yourself performing or behaving in a particular way in a particular situation.

The more often you visualize and imagine yourself acting as if you already had the new habit, the more rapidly this new behavior will be accepted by your subconscious mind and become automatic.
